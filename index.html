Google Sheets Sync

Paste your Apps Script Web App URL below. Once saved, every invoice and expense will sync to your Google Sheet automatically. After saving, open your Google Sheet to confirm data is arriving.

How to set up
  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete any existing code, paste the script from LedgerLite_AppScript.gs
  3. Click Deploy → New deployment → Web app
  4. Set "Who has access" to Anyone → click Deploy
  5. Copy the Web App URL and paste it above
